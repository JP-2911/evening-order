# Evening Order Behavior Pattern System

Purpose:
Identify the underlying function of late evenings.

Principle:
Late nights are not the problem.
The evening is solving something.

Five core patterns:

1. Recovery Evening
User is decompressing from daytime load.
Typical activities:
- Video
- Social media
Function:
Low-effort recovery.

2. Connection Evening
User is maintaining social connection.
Typical activities:
- Chat
- Social interaction

3. Completion Evening
User is finishing unfinished tasks.
Typical activities:
- Work
- Study

4. Emotional Buffer Evening
User is processing emotions.
Typical signs:
- Tired
- Escaping
- Zoning out

5. Late Wind-down Evening
User has not started the wind-down process.

The system identifies patterns.
AI explains them in natural human language.
